# hack-radar
A remix of the data found on the awesome [TW tech radar](http://www.thoughtworks.com/radar)


## Requirements

- git
- lein
- mongodb


## Installation

```bash
$ git clone https://github.com/vise890/hack-radar
$ lein ring server
```
Evaluate `./src/hack_radar/db/seed.clj` to refresh the database.
